## 📱 Using the Calculator

### Basic Operations
- Tap numbers and operators
- Tap `=` to calculate
- Tap `AC` to clear

### Scientific Functions
- `sin`, `cos`, `tan` - Trigonometry
- `log`, `ln` - Logarithms
- `√`, `x²`, `xʸ` - Powers and roots
- `π`, `e` - Constants
- Toggle `DEG`/`RAD` for angle mode

### Memory
- `MC` - Clear memory
- `MR` - Recall memory
- `M+` - Add to memory
- `M-` - Subtract from memory

### History
- Tap history icon (top right)
- Tap any calculation to reuse
- Tap "Clear All" to delete
