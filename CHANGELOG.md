# Changelog

## [1.0.0] - 2024

### Added
- Scientific calculator with full functionality
- Material Design 3 UI with professional styling
- Real-time result preview
- Calculation history with persistence
- Memory operations (MC, MR, M+, M-)
- DEG/RAD toggle for trigonometric functions
- Web and Android support
- GitHub Actions for automated APK builds

### Features
- Standard arithmetic operations
- Scientific functions (sin, cos, tan, log, ln, √, x², xʸ, !)
- Constants (π, e)
- Error handling
- Dark mode theme
- Responsive layout

### Technical
- Flutter 3.0+
- Provider state management
- math_expressions for calculations
- SharedPreferences for persistence
- Web-compatible dependencies
